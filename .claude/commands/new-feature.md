Crée une nouvelle feature complète en suivant ces étapes:

1. **Analyse**
   - Lis les fichiers similaires existants
   - Identifie les patterns à suivre
   - Liste les dépendances nécessaires

2. **Planning**
   - Fichiers à créer/modifier
   - Tests nécessaires
   - Impacts potentiels

3. **Implémentation**
   - Code suivant conventions CLAUDE.md
   - Tests unitaires + intégration
   - Gestion d'erreurs appropriée

4. **Validation**
   - Type-check
   - Linting
   - Tests passent
   - Build réussit

5. **Documentation**
   - Commentaires si logique complexe
   - Update README si nécessaire

Ne crée le commit qu'après validation complète.
