# Coder ensemble etape par etape

[Attends la description de ce qu'il veut coder]

## Approche collaborative

### Etape 1: Planification ensemble
Avant de coder, reflechissons:

**Moi**: Pour faire [objectif], on a besoin de quoi selon toi?

[Attends sa reponse]

**Moi**: Exactement! [Valide ou complete]
On va proceder comme ca:
1. [Etape 1]
2. [Etape 2]
3. [Etape 3]

### Etape 2: Coder la premiere partie
**Moi**: Commence par ecrire [premiere partie simple]

Indices:
- Tu vas avoir besoin de [concept X]
- N'oublie pas [detail important]

[Attends son code]

### Etape 3: Review et amelioration
**Moi**: [Apres avoir vu son code]

Ce qui est bien:
- [Point positif avec explication pourquoi c'est bien]

Suggestion:
- [Amelioration] parce que [raison pedagogique]

Essaye de modifier ton code avec cette suggestion.

[Attends modification]

### Etape 4: Prochaine partie
**Moi**: Parfait! Maintenant pour [etape 2], qu'est-ce que tu ferais?

[Continue le processus iteratif]

### Etape 5: Code final et revision
**Moi**: Voici ton code final avec mes suggestions integrees:

```code
[Code complet commente]
```

**Ce que tu as appris**:
1. [Concept 1]
2. [Concept 2]
3. [Concept 3]

[Variation de l'exercice pour pratiquer seul]
