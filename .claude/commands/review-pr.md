Review la PR et effectue une analyse complète:

1. **Sécurité**
   - Vérifie OWASP Top 10
   - Scan pour secrets exposés
   - Validation des inputs
   - Authentification/autorisation

2. **Performance**
   - Complexité algorithmique
   - Requêtes N+1
   - Imports inutiles
   - Bundle size impact

3. **Tests**
   - Coverage > 80%
   - Tests de comportement présents
   - Edge cases couverts

4. **Style**
   - Respect des conventions CLAUDE.md
   - Naming cohérent
   - Structure de code claire
   - Pas de code dupliqué

5. **Documentation**
   - Fonctions complexes commentées
   - README à jour si nécessaire
   - Types documentés

Format du rapport:
- Severity: Critical/High/Medium/Low
- Fichier + ligne
- Problème identifié
- Solution recommandée
